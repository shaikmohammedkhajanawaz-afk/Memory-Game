Memory Game in Java
📌 Project Overview

Memory Game is a simple console-based Java game where players match pairs of cards by remembering their positions. The cards are shuffled randomly, and the player must find all matching pairs to win the game.

✨ Features
Random card shuffling
Console-based gameplay
Match pair detection
Input validation for card selection
Win message when all pairs are found
🛠️ Technologies Used
Java
ArrayList
Arrays
Collections Framework
Scanner Class
🚀 How to Run
Save the file as MemoryGame.java.

Compile the program:

javac MemoryGame.java

Run the program:

java MemoryGame
🎮 How to Play
The game displays hidden cards.
Enter the index of the first card to flip.
Enter the index of the second card to flip.
If both cards match, they remain visible.
If they do not match, they are hidden again.
Continue until all pairs are matched.
